If this message is visible, it means that an error occurred while reading your student.js file.

  • Open de browserconsole.
    WindowsMacOS
    Chrome Ctrl + Shift + J Cmd + Opt + J
    Firefox Ctrl + Shift + K Cmd + Opt + K
    Safari Cmd + Opt + C
  • The error message should give a hint to what went wrong. Also note the line number on which the error occurs.

If there is no error message or the browser does not seem to respond, it means that you have probably implemented an infinite loop.

  • Remove your most recently written code from the student.js file. The easiest is by commenting it out.
  • Refresh the page. If this fails, close the tab and reopen it (Ctrl + Shift + T of Command + Shift + T).
  • The statements should have reappeared. If not, comment out more code from student.js until everything works again.
  • Review your code and try to figure out why your loop doesn't stop.
